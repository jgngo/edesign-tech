CADSTAR is Zuken's Windows based desktop PCB design solution, easy-to-use, ready-to-use, and easily integrated into your current design environment while preserving your legacy data. It has been developed to take full advantage of the Windows operating system and its inherent benefits, such as the multiple document interface (MDI), object linking and embedding (OLE) and the intuitive Windows graphical user interface (GUI).  This combination results in a CAD system which is easy to learn and use, and which is consistent with other complimentary applications.
